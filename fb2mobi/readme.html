Конвертор командной строки из fb2 в Mobipocket и в Kindle.

Версия 2.0.4 /06.01.2012/
Автор: Sk0rp
Страница проекта: http://code.google.com/p/fb2mobi/

Для работы должен быть установлен .NET Framework 2.0
А также нужен свежий компилятор KindleGen for Windows (2003, XP, Vista, 7), брать отсюда: http://www.amazon.com/kindlepublishing

Использование:
Распаковать fb2mobi.zip, в эту же папку положить загруженный и распакованный kindlegen.exe
Для конвертирования передавать fb2 файл в fb2mobi.exe как параметр.
Например можно кликнуть на fb2 правой кнопкой, выбрать "Открыть как..." и для открытия указать fb2mobi.exe. Можно fb2 файл перетащить мышкой и кинуть на fb2mobi.exe

Ключи командной строки

Usage: fb2mobi <file.fb2> [<output>] [{-,/,--}param]
  -nc    Не сжимать конечный файл. Это увеличивает скорость работы и размер.
  -cl    Удалить папку с промежуточными файлами.
  -us    Использовать имя исходного файла как имя конечного.
  -nt    Не транслитирировать имя конечного файла.
  -V0    Подавлять любой вывод в stdout.
  -V1    Подавлять вывод в stdout. Будет выведена только одна строка с именем конечного файла.
  output имя конечного файла (в этом случае -us и -nt не действуют) или папка для конечного файла*

*ВАЖНО! Если указывается папка для конечного файла, то в конце не должно быть слеша: "\"

Часто задаваемые вопросы:

-Можно ли добавить обработку сразу папки с файлами
-Можно, но лень, это и так легко можно сделать стандартными средствами из командной строки, например:

for %a in (fb2folder\*.fb2) do fb2mobi.exe "%~fa" -cl -nt -us
обработает все fb2 файлы в папке fb2folder, результирующие mobi файлы будут располагаться в этой же папке с теми же именами, что были у fb2 файлов.

->Дайте мне бат файл, который будет обрабатывать все файлы в директории, котрые имеют расширение .fb2 и .fb2.zip.
->А нельзя ли, что бы он и папки и подпапки обрабатывал ?
-Можно, вот так, например:

for /R "%~1\" %%a in ("*.fb2.zip") do "%~dp07za.exe" e "%%~a" -y -o"%%~dpa"
for /R "%~1\" %%a in ("*.fb2") do "%~dp0fb2mobi.exe" "%%~a" -nc -cl
:: Если исходные файлы удалять не надо, закомментируйте следующие 2 строки 
for /R "%~1\" %%a in ("*.fb2.zip") do del /f /q "%%~a"
for /R "%~1\" %%a in ("*.fb2") do del /f /q "%%~a"
bat файл положить в ту же папку, где лежит fb2mobi и kindlegen. Дополнительно скачать и распаковать туда же 7-zip, лучше всего подойдет: 7-Zip Command Line Version 32-bit, брать отсюда: http://7-zip.org/download.html
Папку для обработки передавать bat файлу как параметр.
